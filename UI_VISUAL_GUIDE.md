## Responsive Breakpoints

### Extra Small (Mobile) - < 640px
```
Full width layout
Sidebar: Overlay with toggle

Stacked inputs
```

### Small (Tablet) - 640px - 1024px
```
Full width layout
Sidebar: Overlay with toggle
2-column layout where possible
Side-by-side inputs
```

```
Sidebar takes 256px on left
Main content fills rest
2-3 column layouts
Horizontal forms
```
